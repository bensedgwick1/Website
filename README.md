# Modern Text-Based Adventure Game

This is a modern, mobile-friendly text-based adventure game. The application is built with HTML, CSS, and JavaScript, with a clean separation of concerns.

## How to Play

1.  Open the `index.html` file in a web browser.
2.  The game will start automatically.
3.  Click on the buttons to make choices and progress through the story.
4.  Your character's stats, inventory, and visited places are tracked in the sidebar.
5.  You can save and load your progress using the buttons in the sidebar.

Enjoy your adventure!
